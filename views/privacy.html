Privacy Policy

Last updated: June 8, 2026

1. Data We Collect

Cassie ("the Bot") collects and stores the following data when you use its features:

  • Discord User ID — used to identify you for custom prefixes, reminders, AFK status, todo lists, command usage statistics, chat history, and per-user command overrides.
  • Discord Guild (Server) ID — used to store server-specific settings such as custom prefix, disabled commands, tags, and allow/deny overrides.
  • Discord Channel ID — used for channel-specific command overrides and settings.
  • Discord Role ID — used for role-based command overrides.
  • Command Usage Data — we track which commands you use, in which server, and how often. This is used for statistics and diagnostics.
  • Message Content — when you use chat/AI features, the messages you send are stored temporarily to provide conversational context.
  • User-Created Content — tags, reminders, and todo list items you create are stored so they can be retrieved later.
  • AFK Status — your AFK message is stored to notify others when you are away.

2. Web Dashboard Data

When you log in to the web dashboard via Discord OAuth, we collect:

  • Your Discord User ID, username, and avatar URL
  • Your Discord access token (used only during your session to fetch your guild list)

This data is stored in an encrypted session cookie and is deleted when you log out or after 7 days of inactivity.

3. How We Use Your Data

Your data is used exclusively to provide the Bot's functionality:

  • Processing commands and applying your settings
  • Remembering your custom prefix, reminders, tags, and todo lists
  • Enforcing command restrictions (disable/enable per server, channel, role, or user)
  • Displaying statistics on the web dashboard
  • Improving the Bot based on usage patterns

4. Data Storage & Retention

All data is stored in a PostgreSQL database. We retain data indefinitely to provide continuous service. If you want your data deleted, contact us using the information below. Note that some data may be retained for legitimate operational purposes (e.g., command logs).

5. Data Sharing

We do not sell, trade, or share your personal data with third parties. Data is only accessible to the Bot operator for support and maintenance purposes.

6. Data Security

We take reasonable measures to protect your data, including encrypted database connections and secure session handling. However, no method of electronic storage is 100% secure.

7. Your Rights

You have the right to:

  • Request a copy of the data we hold about you
  • Request deletion of your data
  • Request correction of inaccurate data

To exercise these rights, contact us using the information below.

8. Contact

For privacy-related inquiries, please contact the Bot operator in the support server or send a direct message.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted here and the "Last updated" date will be revised.
